# NovaSound — бэкенд

Node.js (Express), API.

**Переменные окружения:** см. [необязательные переменные.md](./необязательные%20переменные.md) (внутри — и обязательные, и необязательные).

## Запуск локально

```bash
npm install
```

Скопируй `.env.example` → `.env`, заполни MongoDB, JWT, почту и т.д.

```bash
npm start
```

Для разработки с автоперезапуском: `npm run dev` (если стоит nodemon).

## В Git не класть

`node_modules/`, `.env` (секреты).
